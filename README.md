# moyu-framework

A SpringBoot framework for rapid development

MoYu SpringBoot 快速开发脚手架

### MoYu 框架项目结构设计

```
moyu-framework MoYu框架
│ 
├── moyu-autoconfiguers         # 自动装配模块
│  ├── moyu-framework-common    # 公共服务自动装配
│  └── moyu-web-autoconfiguers  # Web开发自动装配
│ 
├─ moyu-base                    # 基础模块（可独立依赖）
│  ├── moyu-model               # 公共模型
│  └── moyu-util                # 公共工具类
│
└─ moyu-sample                  # demo工程
```
